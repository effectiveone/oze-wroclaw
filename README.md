# OZE Wrocław

Energetyczny portal Dolnego Śląska — informacje o odnawialnych źródłach energii we Wrocławiu i regionie.

## Stack
- Astro 4.x (static output)
- Vanilla CSS (vibrant orange/yellow magazine theme)
- Cloudflare Pages

## Uruchomienie lokalne
```bash
npm install
npm run dev
```

## Deploy
Cloudflare Pages: build command `npm run build`, publish directory `dist`
